# 途迹星球（JourneyTrace Planet）- 记录你的每一步旅行
从个人足迹记录到全球旅行分享，你的旅程值得被看见。用地图，写满你的旅行故事，让每一段行走都留下清晰印记，让每一份回忆都能长久珍藏。

## 一、核心功能：让旅行足迹可视化
在途迹星球，你可以轻松将抽象的“去过哪里”转化为具象的地图足迹，通过多维度记录与管理，让旅行回忆不再“看过就忘”。

### 1. 足迹添加：精准标记每一处目的地
添加新足迹时，需完善以下关键信息，确保记录完整准确：
- **基础定位信息**：输入地点名称后可通过搜索精准匹配，系统支持标记经纬度与详细地址，结合高德地图实现精准定位
- **分类标注**：从“景点、博物馆、美食、自然、购物”等类型中选择，明确足迹属性
- **细节补充**：填写前往时间、添加个性化标签（如“亲子游”“徒步打卡”），搭配文字详细描述旅行体验，同时可上传图片或视频还原现场场景
- **隐私设置**：自主选择足迹是否公开展示，私密回忆与公开分享随心切换

### 2. 足迹管理：高效回溯与筛选
- **时间线回溯**：按旅行时间顺序梳理足迹，轻松回顾某一段旅程的完整路线
- **类型筛选**：可按“景点”“美食”等类型快速筛选足迹，精准查找目标记录
- **数据可视化**：系统自动统计旅行天数、覆盖城市数量、总里程，生成专属旅行报告；支持生成旅行热力图，直观展现你走过的山河湖海

### 3. 地图功能亮点
- 高德地图精准定位，确保每一个足迹坐标准确无误
- 支持自定义足迹图标，让地图上的旅行轨迹更具个人风格
- 点击地图即可选择位置标记足迹，按Esc键可快速取消抓点，操作便捷

## 二、星球广场：连接全球同频旅行者
当你想分享旅途喜悦，或为下一站寻找灵感时，途迹星球的“广场”板块（公共足迹区域）为你搭建交流桥梁，让旅行从“一个人的回忆”变成“一群人的热爱”。

### 1. 浏览与借鉴
- 查看其他旅行者公开的足迹，获取不同目的地的攻略与体验分享
- 对心仪的足迹内容进行点赞、收藏，为自己的旅行计划积累灵感

### 2. 分享与帮助
- 发布自己的行程模板，将小众路线、特色玩法分享给更多人，助力他人解锁独特旅行体验
- 参与旅行者间的互动，通过足迹评论交流旅行心得，结识同好

## 三、个人中心：管理专属旅行档案
登录后可进入个人中心，完善个人信息并管理所有足迹记录，打造专属旅行档案：
- **个人资料管理**：更换头像、编辑性别、查看注册时间与最后登录记录，修改密码保障账号安全
- **足迹汇总**：清晰展示已记录的地点数量，支持查看全部足迹或按类型筛选
- **操作权限**：登录用户可进行添加、编辑、删除足迹等操作；未登录游客仅可浏览公共足迹，ID状态为禁用，无法使用记录功能

## 四、立即开启你的途迹之旅
每一段旅程都值得被记录，每一个足迹都值得被分享。现在访问途迹星球（ https://jtrace.lizh.cc/ ），点击“立即开始记录足迹”，用地图标记旅行，让世界见证你的精彩足迹！
